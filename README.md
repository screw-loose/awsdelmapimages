awsdelmapimages
===============================

version number: 0.0.1
author: Rick Bohrer

Overview
--------

Delete map image files on AWS.

Installation / Usage
--------------------

To install use pip:

    $ pip install awsdelmapimages


Or clone the repo:

    $ git clone https://github.com/waxkinetic/awsdelmapimages.git
    $ python setup.py install
